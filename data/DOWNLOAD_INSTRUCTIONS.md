# Dataset Download Instructions

## Dataset: Give Me Some Credit (Kaggle)

**Direct link:** https://www.kaggle.com/c/GiveMeSomeCredit/data

### Steps:
1. Go to the link above
2. Sign in to Kaggle (free account)
3. Click **"Download All"**
4. Unzip and place `cs-training.csv` in this `/data/` folder
5. You can ignore `cs-test.csv` for now

### File you need:
- `cs-training.csv` → 150,000 rows × 11 columns (~25 MB)

> ⚠️ This file is NOT committed to the repo (too large + Kaggle terms).
> The `.gitignore` excludes all `.csv` files in this folder.
